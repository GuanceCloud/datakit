The CockroachDB collector is used to collect CockroachDB-related indicator data.
Currently, it only supports data in Prometheus format.

Already tested CockroachDB version:

- [x] CockroachDB 19.2
- [x] CockroachDB 20.2
- [x] CockroachDB 21.2
- [x] CockroachDB 22.2
- [x] CockroachDB 23.2.4
